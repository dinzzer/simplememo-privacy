Autoflash

プライバシーポリシー

施行日: 2026年3月4日

本プライバシーポリシーは、Autoflash（以下「本アプリ」）をご利用いただく際に、どのような情報が収集・利用されるかについてお知らせするものです。本アプリは個人開発によるもので、ユーザーのプライバシーを尊重し、個人情報の保護に努めます。

1収集する情報

本アプリは、ユーザーアカウントの作成を必要とせず、氏名、メールアドレス、電話番号などの個人情報を直接収集することはありません。

本アプリが扱うデータは以下のとおりです。

端末内に保存されるデータ
  • 登録した単語帳・単語・定義・例文
  • クイズの結果と学習進捗
  • アプリの設定（表示言語の選択など）
  • 広告削除の購入状態

これらのデータはすべて端末内（ローカルデータベース）にのみ保存され、外部サーバーへの送信は行いません。アプリを削除すると、これらのデータも削除されます。

2第三者サービスの利用

本アプリでは、以下の第三者サービスを利用しています。これらのサービスは独自のプライバシーポリシーに基づいて情報を収集する場合があります。

📢
Google AdMob
広告の表示に使用します。AdMobはパーソナライズド広告の配信のために、端末の広告識別子やおおまかな位置情報などを収集する場合があります。アプリ内課金（広告削除）により広告を完全に非表示にできます。
Google プライバシーポリシー →
💳
Google Play Billing
広告削除のアプリ内課金に使用します。購入処理はすべてGoogle Playを通じて行われ、本アプリが決済情報を直接取得することはありません。
Google Play ポリシー →
📖
Free Dictionary API
単語の定義・品詞・例文の取得に使用します。検索した単語がAPIサーバーに送信されます。個人を特定する情報は含まれません。
Free Dictionary API →
🌐
MyMemory Translation API
英語の定義を日本語に翻訳する際に使用します。翻訳対象のテキストがAPIサーバーに送信されます。個人を特定する情報は含まれません。
MyMemory →

3広告について

本アプリでは、Google AdMob による広告を表示しています。広告の配信にあたり、Googleおよびその広告パートナーがユーザーの端末情報や広告識別子を使用してパーソナライズド広告を配信する場合があります。

広告のパーソナライズ設定は、端末の「Google設定」＞「広告」からいつでも変更できます。また、アプリ内課金により広告を完全に削除することが可能です。

4データの保存と管理

本アプリで作成されるすべてのデータ（単語帳、単語、学習記録など）は、ユーザーの端末内にのみ保存されます。クラウドへのバックアップや同期機能は提供していません。

アプリを端末からアンインストールすると、保存されたすべてのデータが削除されます。データの復元はできませんのでご了承ください。

5お子様のプライバシー

本アプリは13歳未満のお子様を対象としていません。13歳未満のお子様から意図的に個人情報を収集することはありません。

6セキュリティ

本アプリは、ユーザーのデータを保護するために合理的な措置を講じています。ただし、インターネット上のデータ送信や電子的な保存方法において、100%の安全性を保証することはできません。

7プライバシーポリシーの変更

本プライバシーポリシーは随時更新される場合があります。変更があった場合は本ページを更新し、施行日を変更いたします。定期的に本ページをご確認いただくことをお勧めします。
